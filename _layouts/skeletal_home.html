Zillion
Make sense of it all

Zillion is a free, open data warehousing and dimensional modeling tool that allows combining and analyzing data from multiple datasources through a simple API. It writes SQL so you don't have to.

Zillion Web UI
Demo UI and Web API

Zillion Web is a monorepo Vue frontend UI and Python backend API with a main objective of demonstrating and testing some use cases for Zillion. Check out the live demo and example Warehouses.

Glide
Easy ETL

Glide is an easy-to-use data pipelining tool inspired by Consecution and Apache Storm Topologies. Glide aims to be a simple, reusable, extensible approach to building ETL pipelines.
